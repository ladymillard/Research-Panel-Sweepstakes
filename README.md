 # Research Panel Sweepstakes

This repository contains official rules and documentation for various GitHub research panel sweepstakes programs.

## Overview

GitHub conducts research with customers to improve products and services. Participants in research studies may be eligible for sweepstakes prizes as a thank you for their time and feedback.

## Available Sweepstakes

This repository includes official rules for the following sweepstakes programs:

- **[GitHub Universe Customer Research Sweepstakes](./GitHub%20Universe%20Customer%20Research%20Sweepstakes%20Official%20Rules.docx.pdf)** - Rules for research participants at GitHub Universe events

- **[Copilot Coding Agent Survey Sweepstakes](./Copilot%20Coding%20Agent%20Survey%20Sweepstakes%20Official%20Rules%202025.pdf)** - Sweepstakes for participants in Copilot Coding Agent surveys

- **[Copilot Coding Agent Churned and Unengaged Survey Sweepstakes](./Copilot%20Coding%20Agent%20Churned%20and%20Unengaged%20Survey%20Sweepstakes%20Official%20Rules%202025.pdf)** - Sweepstakes for churned and unengaged user surveys

- **[Copilot Workspace Survey Sweepstakes](./Copilot%20Workspace%20Survey%20Sweepstakes%20Official%20Rules%202025.pdf)** - Sweepstakes for Copilot Workspace survey participants

- **[Copilot Workspace Churn Survey Sweepstakes](./Copilot%20Workspace%20Churn%20Survey%20Sweepstakes%20Official%20Rules%202025.pdf)** - Sweepstakes for Copilot Workspace churn surveys

## Privacy

All research and sweepstakes programs follow GitHub's privacy policies:
- [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)

## Questions

For questions about specific sweepstakes or research programs, please refer to the contact information provided in the respective official rules documents.
